# ПРОЕКТ КАЛЬКУЛЯТОР

### ФУНКЦИОНАЛ
Для целых чисел
1. Сложение и вычитание - `add(x,y)`, `sub(x,y)`
2. Умножение и деление - `mull(x,y)`, `div(x,y)`
3. Возведение в степень - `pow(x)`
4. Факториал - `factorial(x)`

### ОГРАНИЧЕНИЯ
1. Запрещено использовать `+`, `-`, `//`, `*`, `**`, `%`
2. Запрещено использовать циклы
3. Исключение: разрешено использовать функции `add_one(x)` и `sub_one(x)`.
